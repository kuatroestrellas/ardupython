Air Potato

Manual de usuario


Este sistema aun no esta terminado
por lo que pueden ver es muy simple chafa y facil de usar
esto es solo un holamundo mas adelante el programa se hara
de una forma algo profesional con alguna interfaz mas avanzada
como PyGTK...
algo con muchas funciones que en verdad necesite manual de usuario
y documentacion



El programa monitorea los sensores en tiempo real y almacena los datos
en una base de datos hecha en sqlite cada segundo
tambien notifica con mensajes de texto si algo anda mal
por ejemplo; si se acabo el agua o se destapo la caja de las papas
el sistema de monitorizacion esta escrito completamente en python
se hicieron uso de diversas librerias esternas como:
-pyserial: establece comunicacion con arduino por el puerto serie
-pybluez: comunica con arduino via bluetooth
-python-gammu: controla telefonos celulares... en este caso es el que envia los mensajes
-python-twitter: notifica via twiter
-pyfpdf: crea pdf's
